# Python Apps Collection

This folder contains a small set of beginner-friendly Python command-line applications.

## Projects

### Calculator
- A simple arithmetic calculator that accepts two numbers and an operator.
- Run it from the Calculator folder with `python main.py`.

### Expense Tracker
- A console-based expense manager that allows adding, viewing, editing, deleting, summarizing, and searching expenses.
- Run it from the Expense_Tracker folder with `python main.py`.

### Weather App
- A weather application that fetches data from the OpenWeather API for a user-provided city.
- Run it from the Weather_App folder with `python main.py` after installing dependencies from `requirements.txt`.

## Getting Started
1. Open the project folder you want to use.
2. Install any required dependencies if needed.
3. Run the corresponding `main.py` file.
