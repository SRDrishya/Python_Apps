# Calculator

A simple command-line calculator built in Python.

## Features
- Performs basic arithmetic operations: addition, subtraction, multiplication, division, integer division, modulus, and exponentiation.
- Takes two numbers and an operator from the user.
- Prints the result of the selected calculation.

## How to Run
1. Open the Calculator folder.
2. Run the script:
   ```bash
   python main.py
   ```
3. Enter the first number, second number, and operator when prompted.

## Supported Operators
- `+` addition
- `-` subtraction
- `*` multiplication
- `/` division
- `//` integer division
- `%` modulus
- `**` exponentiation

## Files
- `main.py` – collects user input and displays the result.
- `calculator.py` – contains the arithmetic functions.
